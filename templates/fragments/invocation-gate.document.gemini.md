Full form: {{ROUTE_NAME}} target=<path> ... . A bare path is shorthand for target=<path>. When target= is used, unlabeled paths are rejected.

If a valid target invocation omits mode, missing mode selects read-only. If it omits assurance, missing assurance selects advisory.

Help-style or invalid invocations explain usage only. Do not read the target or reference bodies, do not run workflow commands, do not run probes, do not create state, and do not declare a review result.

Before any drfx workflow command, materialize <selectedGuard> from explicit guard or default git, and pass guard=<selectedGuard> to the workflow command.
